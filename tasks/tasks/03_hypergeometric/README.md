# Задача 3. Гипергеометрическое распределение

**Условие**  
В партии 25 микросхем, из них 6 - бракованные.  
Наудачу отбирают 4 микросхемы.  

**Требуется**  
а) составить закон распределения ДСВ;  
б) найти математическое ожидание, дисперсию, среднеквадратичное отклонение, моду;  
в) определить вероятность отсутствия бракованных микросхем;  
г) построить многоугольник распределения.
