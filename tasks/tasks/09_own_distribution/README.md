# Создание собственных распределений в Python
Дана функция плотности НСВ.
Для этой случайной величины:
- проведена проверка условия нормировки плотности;
- построены графики плотности и функции распределения;
- рассчитана вероятность попадания случайной величины в некоторый интервал;
- вычислены математическое ожидание, дисперсия и среднее квадратическое отклонение; 
- вычислены квантиль уровня q и p%-ная точка случайной величины;
- определён коэффициент асимметрии и эксцесс.
