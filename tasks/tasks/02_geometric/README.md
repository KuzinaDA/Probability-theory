# Задача 2. Геометрическое распределение

**Условие**  
Вероятность обнаружения дефекта при проверке изделия равна p = 0,1.  
Проверка производится до первого обнаружения дефекта.  


**Требуется**  
а) составить закон распределения ДСВ;  
б) найти математическое ожидание, дисперсию, среднеквадратичное отклонение, моду;  
в) определить вероятность того, что потребуется проверить не менее 6 изделий.
г) построить многоугольник распределения.
