# Задача 4. Распределение Пуассона

**Условие**  
Число аварий на участке дороги за 1 день — случайная величина X, имеющая распределение Пуассона с параметром lambda = 1,5.

**Требуется**  
а) составить закон распределения ДСВ X при X от 1 до 9;  
б) найти математическое ожидание, дисперсию, среднеквадратичное отклонение, моду;  
в) вычислить вероятность того, что за 1 день произойдёт более одной аварии;  
г) построить многоугольник распределения.
